## 扩展功能

### 命令行启动参数

在应用程序启动时，可以使用命令行参数设置一些功能。

| 参数     | 说明        | 用例               |
| ------ | --------- | ---------------- |
| vendor | 指定运行的虚拟环境 | -vendor=myvendor |

windows平台可在程序的快捷方式的属性中，`目标`一栏的最后追加参数，例如

```
fmp.exe -vendor=myvendor
```



### 隐藏式退出

以小于1秒的间隔快速点击15次设备码，在设备码消失后的1秒内以小于1秒的间隔快速点击15次二维码，即可退出应用程序。
